## 项目定位

**ScholarAgent** 是一个基于 ReAct 推理框架的智能学术研究辅助工具，以微服务架构组织后端能力，能够自主规划检索、阅读、总结与引用生成任务，为用户提供端到端的学术问答体验。

### 核心价值

- **智能推理**：ReAct + Function Calling，透明可观测的思考链
- **学术专项**：论文搜索、PDF 解析、RAG 知识库、BibTeX 引用
- **工程示范**：Go 微服务、gRPC、容器化部署、Prometheus 可观测性

## 技术选型总览

| 类别 | 技术选型 | 理由 |
| ---- | -------- | ---- |
| 主力 LLM | DeepSeek-V4-Flash | 284B/MoE、100 万 token 上下文、FC 原生支持、性价比高 |
| 备用 LLM | MiMo V2.5 Pro / MiniMax-M3 | Token Plan 可用，降级备选 |
| ReAct | Function Calling + JSON Mode 兜底 | FC 稳定性最优 |
| 向量数据库 | Qdrant | 过滤查询丰富，Go SDK 成熟 |
| 会话存储 | Redis | 高性能，TTL 自动过期 |
| 异步任务 | RabbitMQ | PDF 解析等耗时任务 |
| OCR/PDF | 百度 OCR MCP（主）+ PaddleOCR MCP（备） | 云端快速 + 本地私有化 |
| 可观测性 | Prometheus + Grafana | Pull 模式，适合 K8s |
| 部署 | Docker Compose + Kubernetes（Minikube） | 本地开发 + 生产化演示 |
| 前端 | React + Vite + TailwindCSS + shadcn/ui | 快速构建思考链 UI |
